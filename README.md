# DnsBlade
Quick DNS Zone Transfer

# Installing  And Running DnsBlade

`git clone http://github.com/SxNade/DnsBlade`

`cd DnsBlade`

`pip3 install termcolor; pip3 install dnspython`

 `python3 dnsblade.py <Domain>`
